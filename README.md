About the project
-------------
**Courantderacines** is a simple page application build in react and using many other frameworks.


Make it work
-------------

#### Install and dependencies
```
git clone https://github.com/EKlevrai/courantderacines.git
cd courantderacines
npm install
```
#### Run
```
git clone https://github.com/EKlevrai/courantderacines.git
cd courantderacines
npm start
```
